ASL (American Sign Language) To Text Translator

Developed a Transformer Model to translate ASL gestures into English text using Python, OpenCV, MediaPipe, and C++

The goal of this project is to develop a machine learning model capable of translating American Sign Language (ASL) gestures into written English text in real time. This solution aims to bridge the communication gap between Deaf/hard-of-hearing individuals and non-ASL speakers, providing accessibility and inclusivity in various contexts. While many resources exist for text or speech-based communication, there is a significant lack of real-time, automated tools for ASL users to convert their signed gestures into text. Existing solutions often require expensive hardware or are not robust enough for diverse environments. Our objective is to build an accessible, accurate, and user-friendly ASL-To-Text model using commodity hardware (e.g., webcams or mobile phone cameras).

Home Lab Infrastructure

Architected and deployed a complete virtualization and services infrastructure using Proxmox, managing game servers, databases, DNS filtering, and home automation

Designed and implemented a comprehensive home lab environment leveraging Proxmox as a hypervisor to manage multiple virtual machines and services. The infrastructure hosts video game servers via Pterodactyl for game management and deployment, runs TrueNAS for reliable network-attached storage and data redundancy, and uses Pi-hole for network-wide ad blocking and DNS filtering. Additionally, the lab includes Home Assistant for home automation, CockPIT for streamlined server management and monitoring, and a PostgreSQL database server that powers backend APIs for custom Android and iOS applications.
